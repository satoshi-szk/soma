# AGENTS

## 注意事項

- PR を出す時は main に向けて出して。
- 特に python 側はテストを書いて、自分で動作確認してください。
- テストは、動作確認に必要な、最小限の重要テストだけを記述してください。網羅的なユニットテストは不要です。
- 動作ログは logs 以下にあります。ユーザーに確認せず、自分で読んでください。
- GUI に表示される文言は英語でお願いします。コード内コメントなどの開発者向け文言は日本語でお願いします。

## 作業完了時のルーティン

Python 側のコードを変更した際は、作業完了前に以下を確認してください。

```sh
uv run pytest
uv run mypy src
uv run ruff check .
```

Frontend 側のコードを変更した際は、作業完了前に以下を確認してください。

```sh
npm run build
npm run lint
```
